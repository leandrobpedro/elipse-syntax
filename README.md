# Elipse Software language support in Sublime Text

Add syntax highlighting and snippets to Elipse E3 and Elipse Power scripts in Sublime Text based on VBScript.

## Install
* Download and install `Sublime Text 3`: https://www.sublimetext.com/3</br>
* Download the last version of Elipse Syntax from: https://github.com/leandrobpedro/elipse-syntax/releases/</br>
* Unzip the file in the following path: `%APPDATA%\Sublime Text 3\Packages\User\`</br>
* Open Sublime Text, go to `View >> Syntax >> User >> Elipse` or simple `Ctrl + Shift + P` and type `Elipse`
